# An XFCE Desktop in Docker

- Build the image using `sh build.sh`

- Run an instance using `sh start.sh [id]`
  The default id is 1, which listens on port 4001 (rdp) and 5901 (vnc)
  
- Connect to your instance using a remote desktop desktop

- Default login password is 'xfce' (no username is required)
